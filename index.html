What is BNbitcoin?

BNbitcoin is the first pure mined PoW cryptocurrency on the Binance Smart Chain network. It is a tokenized version of the original bitcoin project by Satoshi Nakamoto (whitepaper).
It derives from the 0xBitcoin project already available on the Ethereum blockchain, but with some features that make BNbitcoin unique and closer to the original bitcoin design.
As bitcoin, there is no airdrop nor ICO: it can only be mined and traded on the exchanges.

FAQ

How can I obtain it?

You can obtain BNBTC mainly by mining it, but it is recently available also on some exchanges. BNBTC liquidity increases with mining, which increments BNBTC availability on the network.
You can mine it both with CPU or GPU, using a common desktop computer or a laptop. Gaming PCs are very well suited for mining BNBTC now.

Can you provide more details about mining BNbitcoin?

To start with BNBTC we suggest to install Metamask to create an account (exactly like in the case of an Ethereum account), and add the Binance Smart Chain to your networks portfolio, as described here.
Now, if you want to mine BNBTC, you have to fund your account with BNB on the Smart Chain mainnet: this account will be your mining account. Funds are needed because of the (low) gas-fees you have to pay when you mint BNBTC tokens after finding a solution and claim the reward (when you mine BNBTC using your devices and you submit the solutions interacting directly with the smart contract, it is called "solo-mining").
Solo-mining is now the only way to mine BNBTC (no pools are available yet).
BNBTC can be mined with moderate hardware resources - even a regular laptop finds solutions from time to time. The best option is a gaming PC with AMD or CUDA-enabled Nvidia GPU. There are no pool fees, and all concurrent solo miners compete against each other. If two or more miners find a complete solution for the same block, the one with the highest gasPrice gets the tokens. For others, mint transaction fails and a negligible amount of BNB gets burned.
Remember that mining will not yield tokens if there are no BNB funds when new solutions are found. Keep an eye on the BNB balance of your mining address while you're mining.
You can choose your favourite mining software from the links above and follow the setup guides reported below.

Why should I mine BNbitcoin instead of other popular mineable tokens on Ethereum?

Because after the DeFi massive widespread on the Ethereum mainnet, gas fees for transactions have become too high. This has a relevant and heavy impact on all the transactions involving all the projects, and particularly if considering mineable tokens: minting new coins and claiming rewards have become too expensive and often unprofitable.
The alternative is offered by the Binance Smart Chain: with its low gas fees, fastness and smart-contract execution capabilities, it is the most suitable environment to give rebirth to the bitcoin protocol through the tokenized minable token BNBTC.

What is the Binance Smart Chain?

Binance Smart Chain is a Proof-of-Staked-Authority (PoSA) blockchain which enables the creation of smart contracts. Its native currency is BNB, it is EVM-compatible and it creates an ecosystem where validators, token holders, developers, and users all benefit from a rewarding blockchain that offers high performance and ample space for further innovations. It has very low transaction fees and blocks are generated every 3 seconds, making this environment the most suitable for the recent DeFi-revolution applications and for mining and trading BNBTC without worrying too much about the minting/transaction fees.

Can I trade BNbitcoin on the exchanges?

Yes! You can trade BNBTC now on DEXes like PancakeSwap, ... (see links below) but works are in progress to have BNBTC listed also on other exchanges.

Has the BNbitcoin smart contract been audited?

The original smart contract of 0xbitcoin has been subject of a Community audit and it did not reveal any relevant issues. The BNbitcoin contract differs for some numerical settings (to better fit to the Binance Smart Chain block time and to restore some parameters as in the original Satoshi Nakamoto's bitcoin) and for an additional read-only function to make it compatible with BEP-20 standards and which is necessary for binding with BEP2 token of the Binance Chain. For this reason the BNbitcoin smart contract source code can be considered equivalent to 0xbitcoin in terms of safety. 

Mining Setups

 COSMiC Miner

Please note that mining software may extract a modest developer fee from time to time; Cosmic has a 2% minimum fee, but you can increase it if you like

Your need a Windows PC.
Check that you have a CUDA-enabled Nvidia GPU. Use this free utility to check your equipment and drivers.
Update your Visual C by running vc_redist.x64.exe update from Microsoft (optional but recommended).
Download Cosmic Miner if you didn't it before, and vcruntime140_1.dll (if needed).
Extract all in one folder.
Launch Cosmic.exe application.
Create a shortcut to Cosmic.exe on your desktop (optional).

Configuration (Windows only, solo-mining):

- Open Cosmic Miner (Cosmic.exe). Go to "Options" -> "Solo Mining Setup."

- Paste the following RPC URL in "Node Address:" field: https://bsc-dataseed.binance.org (or other valid BSC RPC URL from here)

- Paste BNBTC contract address 0xE7Cb24F449973D5B3520E5b93D88B405903c75Fb in "Contract Address:" field (if missing).

- "ChainID" must be set to 56.

- Check that "Gas Price (Gwei):" is at least = 10. 

- Import your mining address private key and encrypt it using a password. Save the password for convenience (optional). Your private keys stays on your PC.

- Click "Save settings."

- From drop-down (bottom right) Select "Solo Mode."

- Hit "Start Mining!"

- Get solutions and newly mined BNBTC tokens!

 MVis-tokenminer

This setup guide allows you to customize MVis-tokenminer for BNBTC solo-mining.
It should work with CPU and with any GPU that supports OpenCL, ie. pretty much all AMDs and most NVidia.
Prepackaged binaries are available for Windows; for Linux files must be compiled from source.

Configuration (Windows only, solo-mining):

- Download and install both the VC 2013 Redistributable and the VC 2015 Redistributable from Microsoft website

- Unzip the MVis-tokenminer binaries download package anywhere you like.

- Open up tokenminer.ini  file using any text editor and set the following configuration items:

   Under [node] section:
   Host=https://bsc-dataseed.binance.org   [or other valid BSC RPC URL from here]
   RPCPort= [leave this field empty]

   Under [0xBitcoin] section:
   MinerAcct= [Your mining address, which starts with 0x and contains BNB to interact with BSC]
   AcctPK= [The private key of your mining address, needed to submit solutions to the contract]

   TokenContract=0xE7Cb24F449973D5B3520E5b93D88B405903c75Fb 
   
   GasPrice=10 [ore more, depending if there ae concurrent miners]

   GasLimit=100000

There are also other settings to control the GPU temperature while mining. It is important to not stress GPU with high temperatures. See your GPU specifications to set such parameters within the nominal working temperatures / thermal loads.

To start solo-mining with GPU you can enter in a command line prompt: .\tokenminer.exe -S -G
To start solo-mining with CPU you can enter in a command line prompt: .\tokenminer.exe -S -C

A help section is available entering: .\tokenminer.exe --help

Further setup details are available on the MVis-tokenminer GitHub page

 SoliditySHA3 miner

A GUI-based release is available here. For the settings please follow the setup description below.

Windows & Linux solo-mining configuration using SoliditySHA3Miner: 

0) Prequisites :

0a) .Net Core Runtime
* Windows => download and run
* Linux => run script 'sudo ./install-deps.sh'

0b) Nvidia driver, and/or AMD driver.

1) Ensure that you have sufficient BNB stored in your mining address (BNB is required to mint tokens).

2) Clone 0xbtcSolo.bat (for Windows) or 0xbtcSolo.sh (for Linux) file and open the editor for the cloned file.

3) You only need to edit the line that starts with 'dotnet' for launch parameters.

4) Below are the following arguments that may/need to be changed :

4a) web3api=https://bsc-dataseed.binance.org
* Other providers are available here. 

4b) abiFile=
* If omitted, the '0xBTC.abi' file will be used by default; it works fine with BNBTC.
* Replace the value to the available filename that ends with '.abi' in the folder for the token interface you are mining.

4c) contract=0xE7Cb24F449973D5B3520E5b93D88B405903c75Fb
* This is the BNBTC smart contract address on BSC. Do not alter it (the software is case-sensitive).

4d) gasToMine=10
* Replace the value with the amount of gas you prefer to mint a block with (decimals allowed, no alphabets).

4e) gasApiURL=

4f) gasApiPath=

4g) gasApiMultiplier=

4h) gasApiOffset=
* Depends on how much you want to offset (decimals allowed, no alphabets).

4i) privateKey=
* Replace the value with your own BSC private key (64 characters long excluding '0x' prefix)
* It is your own responsibility to keep your private BSC account safe.
* One suggestion is to create a disposable BSC mining account to mine with then transfer the tokens to your private account later.

5) Save and (run the '.bat' file for Windows)/(sh the '.sh' file for Linux) you have newly created.

Notes: 

To withdraw your tokens, you will need BNB to perform withdrawal/transaction from your account.

There is a default of 2.0% dev fee (Once every 50th nounces: starting from 50th).
* You can set to the lowest 1.5% with 'devFee=1.5' (the formula is '(nonce mod devFee) = 0').
* Dev fee is by sending the current reward amount after the successful minted block, using the same gas fee as provided in 'gasToMine'. 

 Azlehria's gpuminer

This mining software is a command line tool which allows both CPU and GPU mining with a few simple prompt commands.

NOTICE: from preliminary tests on BNBTC, an earlier version is suggested in this moment for solo-mining.
The following sequence of prompt commands (CMD.exe shell) is an example for solo-mining with a Nvidia GPU within a Windows environment:

>.\0xbitcoin-miner.exe
[this commands opens the mining software instance, but it does not start mining: the following commands need to be entered for the mining setup]

>help
[this commands shows the available options and other commands of the mining software]

>account new
[it creates a new account, which is empty, and you must fill with BNB to submit solutions to the contract. You'll be provided also the private key of the account. You can also mine with the empty account, without BNB funds, but the found solutions won't be submitted to the BNbitcoin smart contract]

> account list
[a list of available accounts, including the account created above]

>config list
[this command shows the current configuration setup]

>config gasprice 10
[this command sets the gasPrice to 10 BNB-Gwei]

>config web3provider https://bsc-dataseed.binance.org
[this command is required to link the miner to the Binance Smart Chain]
 
>contract select 0xE7Cb24F449973D5B3520E5b93D88B405903c75Fb
[this commad is required to point the miner to the BNbitcoin smart contract address on BSC]

>mine cuda
[this command starts mining! :) ]
